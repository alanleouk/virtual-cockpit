Virtual Cockpit Homepage

A quick and dirty project to server Microsoft Flight Simulator (MSFS) cockpits to local or remote machines.

Find the source code on GitHub virtual-cockpit

Quick Setup (WIP)

Install the WASM driver in MSFS. See What Is A 7Z File (And How Do I Open One)?

Load the virtual cockpit web application.

Run the virtual cockpit host application on machine that is running MSFS.

Configure the windows hosts file to resolve 'vcockpit-host.osz.one' to the host machine IP on the client running the cockpit. See How to Find Your Private and Public IP Addresses and How to Edit Your Hosts File on Windows, Mac, or Linux.
